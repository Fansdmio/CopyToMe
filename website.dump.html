使用方法

1. 输入内容

在两个 . 之间写入需要处理的内容

2. 复制等待

复制内容至剪贴板，等待AI处理完成

3. 粘贴答案

中文模式下按「v→2→空格」调出结果

4. 必备配置

需提前安装并启用微信输入法

核心特性

  • 突破应用网页禁止粘贴限制
  • 操作简单，仅需基础复制粘贴步骤
  • 剪贴板自动处理，无需额外操作

使用前提

  • 仅支持 Windows 系统
  • 设备需安装 微信输入法
  • 粘贴时需保持输入法为 中文模式
